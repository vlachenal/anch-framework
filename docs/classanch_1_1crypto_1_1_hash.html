Detailed Description

template<std::size_t O, std::size_t B>
class anch::crypto::Hash< O, B >

Hash algorithm abstract class.

N is the hash output size in bytes.
B is the hash block size in bytes (used in HMAC algorithm).

Vincent Lachenal
