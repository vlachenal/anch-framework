Detailed Description

Cryptography namespace.

It contains classes for cryptography facilities:

  • Hash functions (MD5, SHA1 and SHA2)
  • Cipher functions (AES)
  • Message authentication code functions (HMAC)
  • Binary to text functions (Base64)
  • Block cipher mode of operation (ECB, CBC, PCBC, OFB, CFB, CTR)
  • Padding algorithms (PKCS5Padding, ZeroPadding, ISO7816_4Padding, ANSIX923)
