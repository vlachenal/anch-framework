Detailed Description

Date string formatter.

Paterns are:

  • %Y: 4 digits year
  • %y: 2 digits year
  • %m: Month (1-12)
  • %d: Day in month (1-31)
  • %H: 00-23 hours
  • %h: 00-11 hours
  • %p: AM/PM marker
  • %M: minutes (00-59)
  • %S: seconds (00-59)
  • %s: milliseconds (000-999)

Do not use characters '%' in your string pattern.

Warning: This class is not thread safe.
